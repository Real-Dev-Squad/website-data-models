# Progress updates

## Firestore Collection

- progress

```js
{
    task_id: string
    user_id: string
    progress: string
    plan: string
    blockers: string
}
```

## Example

```js
{
    task_id: "ExjhruG94nghd"
    user_id: "dhgei43ijew9s"
    progress: "I have completed finalizing the data model and raised a PR"
    plan: "Before the next update i am planning to raise PR for the frontend"
    blockers: "None"
}
```
